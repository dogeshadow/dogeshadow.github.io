P5705 数字反转

2021-02-07- 2021-02-08
编程题 > Luogu in C-C

题目描述

输入一个不小于 100 且小于 1000，同时包括小数点后一位的一个浮点数
要求把这个数字翻转过来并输出

输入格式

一行一个浮点数

输出格式

一行一个浮点数

输入输出样例

输入 #1

123.4

输出 #1

4.321

题解

法一

因为题目限定了格式，所以直接按照格式 scanf 每一位数再反过来输出

#include <stdio.h> 

int main(void)
{
	char a, b, c, d;
    
	scanf("%c%c%c.%c", &a, &b, &c, &d);
	printf("%c.%c%c%c", d, c, b, a);
	
    return 0;
}

法二

把数当作字符串处理，从后往前输出每一个元素，包括小数点
计算字符串有效长度 (不包括 ‘\0’) 的方法：使用 <string.h> 头文件里的 strlen 函数
注意：翁恺老师讲的 sizeof(a) / sizeof(a[0]) 检测的是数组 a 的长度
本题这种初始化方式 (先声明再 scanf) 采用这种方法得出的长度是声明的数组长度 10，根本无法计算字符串有效长度
即使采用可以计算的初始化方式 (e.g., char a[] = "Hello")，这种方法得出的也是包括 ‘\0’ 的字符串长度 (6)

#include <stdio.h>
#include <string.h>

int main(void)
{
    char num[10];
    int len;
    
	scanf("%s",num);
    len = strlen(num);
    
	for( len = len - 1; len >= 0; len-- )
    	printf("%c",num[len]);
    
	return 0;
}

法三

把数当作数处理，得到每一位数，再按照题目规定顺序输出

#include <stdio.h> 

int main(void)
{
	float num;
	int d, intnum, out;

	scanf("%f", &num);	
	d = (int)(num * 10) % 10;
	printf("%d.", d);

	intnum = (int)num;
	while (intnum) {
		out = intnum % 10;
		printf("%d", out);
		intnum /= 10;
	}

	return 0;
}
