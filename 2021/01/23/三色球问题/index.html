三色球问题

2021-01-23- 2021-02-03
python编程题-python

题目描述

有红、黄、蓝三种颜色的球，其中红球 3 个，黄球 3 个，绿球 6 个。

先将这 12 个球混合放在一个盒子中，从中任意摸出 8 个球，编程计算摸出球的各种颜色搭配。

输入输出格式

输入格式

该题无输入

输出格式

设共有n种颜色搭配，则输出n行，每行输出一种颜色搭配。

每一行依次输出红球个数，黄球个数，绿球个数，并用逗号+空格隔开。

题解

设摸出的红球个数为red，黄球个数为yellow，绿球个数为green

根据题目条件“摸出八个球”，构建等式red + yellow + green == 8

然后根据题目条件限定球的个数。(green >= 2)

for red in range(4):    # 可以取0-3
    for yellow in range(4):    # 可以取0-3
        for green in range (2,7):    # 注意从2开始
            if red + yellow + green == 8:
                print(red, yellow, green)

另外一种解法，思路差不多，代码实现不太一样

for red in range(4):
    for yellow in range(4):
        green = 8 - red - yellow
        if green <= 6:
            print(red, yellow, green)
